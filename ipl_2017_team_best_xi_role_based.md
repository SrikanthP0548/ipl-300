# IPL 2017 — Team-wise Best Possible XI

This file contains only the role-based Best Possible XI for each team. It does not include the full master batting or bowling tables.

## Selection conditions used

- Each XI is selected from that team only.
- Each XI is role-balanced: 2 openers, middle order, finisher(s), wicketkeeper, all-rounder(s), spinner(s), and pacer(s).
- Selection is not based on raw runs/wickets alone. A player can miss out if picking him breaks team balance.
- The XI table keeps batting and bowling statistics side by side.
- Empty bowling cells mean the player did not have a bowling record in the supplied 2017 bowling table.
- Empty batting cells mean the player did not have a batting record in the supplied 2017 batting table.
